# jQuery.validator.setDefaults()

Modify default settings for validation.

## Description

Accepts everything that [validate()](/validate) accepts.

## Usage

**options** *(Object)*
: Options to set as default.

## Examples

Sets the debug setting for all validation calls.

```javascript
jQuery.validator.setDefaults({
	debug: true
});
```
